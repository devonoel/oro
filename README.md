# Oro

A little game made in Lua/Love2D, with minimalism in mind.

## How to play
+ `space` to start
+ `esc` to quit
+ `g` to grow in size
+ `r` to reduce in size
+ Arrow or WASD keys to move

The objective is to get the snake to eat its own tail. If it eats any other part of its body, its game over.
